# Security Policy

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub issues.**

Email exorrnull@gmail.com with:
- Type of vulnerability
- Full path of source file(s) related
- Step-by-step instructions to reproduce
- Proof-of-concept or exploit code
- Potential impact

We acknowledge within 48 hours. Detailed response within 7 days.

## Responsible Disclosure

Valid security reports are credited in our security advisories.
